css计数器

发表于2025-08-22|更新于2025-08-22|前端CSS
|浏览量:

CSS 可以通过其计数器功能实现不使用 js 在列表前添加序号等操作。

1. 初始化计数器

  • name: 设定的计数器名称
  • number：计数器初始值，默认为 0

2. 操作值

  • name: 计数器名称
  • number: 增加的值，默认为 1，可以设置为 0 和负数

3.获取并显示值

  • name: 计数器名称
  • style: 计数器样式，见 counter-style_计数器样式

也可使用 counters

  • name: 计数器名称
  • string: 用来定义多个计数器同时使用的连接符
  • style: 计数器样式，
