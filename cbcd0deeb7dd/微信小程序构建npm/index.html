在微信小程序中初始化 npm

从小程序基础库版本 2.2.1 或以上、及开发者工具 1.02.1808300 或以上开始，小程序支持使用 npm 安装第三方包。

初始化 npm

  1. 在小程序项目根目录下执行 npm init 命令，生成 package.json 文件。

    如果仅需要初始化而暂时不安装依赖，可以在 package.json 目录下创建一个空的 node_modules 目录。

  4. 打开”工具”->”构建 npm”，点击“确定”开始构建。
