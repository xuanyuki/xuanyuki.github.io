hexo配置aPlayer播放器

发表于2025-08-14|更新于2025-08-14|hexo
|浏览量:

首先安装 hexo-tag-aplayer 插件 npm install hexo-tag-aplayer --save

然后在主题配置文件 _config.**.yml 文件中添加配置

之后根据官方文档进行配置即可 hexo-tag-aplayer

这里主要介绍配置aPlayer全局播放的配置方式

首先设置 asset_inject 为 false，之后在主题的配置文件中使用 inject 注入aPlayer的dom结构，由于需要全局播放所以需要开启 pjax 功能。注意 pjax 和 aplayer 配置项在同一个配置文件中最好。

aplayer使用的参数如下：

选项 默认值 描述
data-id 必填 音乐id、播放列表id、相册id、搜索关键字
server 必填 音乐数据源：netease（网易云）、tencent（QQ音乐）、kugou、xiaomi、baidu
type 必填 song（音乐）、playlist（播放列表，歌单）、album、search（关键词搜索）
fixed false 是否开启固定模式
mini false 是否开启迷你模式
loop all 列表循环模式：all,one,none
order list 列表播放模式：list,random
autoplay true 是否自动播放,根据浏览器规则可能不可用
volume 0.7 初始音量，范围为0到1
preload auto 预加载模式：none,metadata,auto
theme #ad7a86 主题颜色，支持16进制颜色代码
listmaxheight 340px 播放列表最大高度，单位为px
mutex true 互斥模式，开启后同一时间只能播放一个播放器实例
listfolded false 列表折叠状态，默认不折叠

aPlayer 官方文档：APlayer
